# distantjetlines' Clean Texture Collection


A collection of PBR enabled testchamber textures for Portal 2: Community Edition. All the basics you should need.

Provided in 512px/128u resolution for performance and to avoid harsh clashing with non-updated assets.

Credit required if used.

### Textures included:
* Black testchamber tiles
* White Testchamber Tiles
* Testchamber glass with lines and without
* Observation room textures
* White plastic trim
* Backanels in gray and yellow, both 64 and 128 sizes
* Squarebeam brush textures in gray and blue
* BTS plastic wall in grey w/o stickers



Technical details: RGBA8888 w/ ZSTD lvl22. Many textures have parallax heightmaps embedded, but parallax disabled for performance reasons and personal visual preferences. 

For those textures, VMT files have parallax feautures commented out. If you choose to enable them, accept that you might have to tweak those settings to achieve a desirable result.
